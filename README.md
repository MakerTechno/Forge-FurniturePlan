# 写在开头
我是MakerTechno，一位高中生。

我在15岁那年创立了这个模组库，比这更早的是TerrariaMod，现在已被资源化。

我已经学了4年的模组，断断续续的，所以即使现在技术也不怎么好。

总之，希望你能在这里体验美好生活 :D

## 使用方法
下载对应版本的`FurniturePlanMod`并将其放置于版本文件夹下`mods`文件夹，然后启动游戏。

现在只支持`Minecraft 1.19.4`使用`Forge`加载器和`Minecraft 1.21`使用`NeoForge`加载器。
